# Extras

Bu klasor, GitHub Pages tarafinda calismayan ama proje kapsaminda saklanmak istenen kod ve dosyalar icindir.

## Klasorler

- `extras/firmware/`: ESP/MCU firmware kaynak kodlari
- `extras/backend/`: sunucu/API, PM2 veya servis tarafi kodlar
- `extras/archive/`: aktif olmayan, referans veya eski deneysel dosyalar

Not: Bu klasordeki dosyalar frontend calisma akisini etkilemez.
